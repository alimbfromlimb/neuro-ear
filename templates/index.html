About the project
The goal of NEURO-EAR is to design the best machine learning algorithm for the problem of classification of musical instruments. Here we are comparing different approaches, including convolutional neural networks, deep CCA, etc. You can upload your own files and see which of the algorithms performed the best. We truly hope to develop the accurate, state of the art model, so your feedback will be highly appreciated
Instrument classification using Deep CNN
See the pipeline we developed to classify instruments using CNNs:
Fourier Transform
A given a .wav file is first converted to a spectrogrm
CNN
Then each one-second piece of the spectrogram goes through a deep convolutional neural network
Generalization
The result is generalized using information about every chunk go the file
Now, choose your own file to be classified with the net!
